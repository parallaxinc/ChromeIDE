Welcome to the Parallax IDE Installer from Parallax Inc.

Introduction

Parallax IDE is a BASIC Stamp editor.

It allows development for the BASIC Stamp microcontroller.

This software installs the Parallax IDE software into your Applications folder.

The required FTDI USB Serial Driver will also be installed. The system must be restarted after installation to use it.
